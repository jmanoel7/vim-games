# vim-games
vim games: tetris, sokoban and sudoku

to install in ~/.vim/bundle with pathogen:
```sh
cd ~/.vim/bundle
git clone git://github.com/jmanoel7/vim-games
```

to install in ~/.vim/bundle with vundle, put it into ~/.vimrc:
```sh
Bundle 'jmanoel7/vim-games'
```
and execute in vim:
```sh
:BundleInstall
```
